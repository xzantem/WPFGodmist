Detailed Description

Efekt umiejętności, który przywraca punkty zasobów (np. many) celowi.

Może być używany do regeneracji zasobów zarówno sojusznikom, jak i przeciwnikom. Wartość regeneracji może być obliczana na podstawie różnych parametrów.
