Detailed Description

Efekt umiejętności, który przełącza (włącza/wyłącza) efekt pasywny nasłuchujący zdarzeń.

Umożliwia dynamiczne włączanie i wyłączanie efektów, które reagują na określone zdarzenia w walce. Jeśli efekt jest aktywny, zostanie wyłączony, a jeśli nieaktywny - zostanie włączony.
