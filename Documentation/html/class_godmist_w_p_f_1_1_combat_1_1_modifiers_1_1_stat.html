Detailed Description

Klasa reprezentująca statystykę postaci, która może być modyfikowana przez różne efekty.

Zawiera wartość bazową, współczynnik skalowania i listę modyfikatorów. Wartość końcowa jest obliczana na podstawie poziomu postaci i aktywnych modyfikatorów.
