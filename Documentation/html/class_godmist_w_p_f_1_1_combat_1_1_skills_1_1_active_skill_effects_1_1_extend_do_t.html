Detailed Description

Efekt umiejętności, który przedłuża czas trwania efektu DoT (Damage over Time) na celu.

Umożliwia przedłużenie czasu trwania obrażeń odnoszonych z upływem czasu, takich jak krwawienie, zatrucie czy oparzenia.
