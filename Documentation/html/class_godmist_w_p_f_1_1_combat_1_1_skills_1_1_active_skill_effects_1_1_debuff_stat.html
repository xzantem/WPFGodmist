Detailed Description

Efekt umiejętności, który zmniejsza wartość wybranej statystyki celu.

Może być używany do osłabiania przeciwników lub samookaleczania się. W przypadku używania na przeciwniku, uwzględnia jego odporność na efekty osłabiające.
