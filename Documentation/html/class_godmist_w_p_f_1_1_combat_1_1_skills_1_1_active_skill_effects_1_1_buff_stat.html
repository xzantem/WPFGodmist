Detailed Description

Efekt umiejętności, który zwiększa wartość wybranej statystyki celu.

Może być używany do wzmacniania własnych statystyk lub osłabiania przeciwnika. Wspiera różne typy modyfikatorów (addytywny, mnożnikowy itp.).
