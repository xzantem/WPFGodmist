Detailed Description

Abstrakcyjna klasa bazowa reprezentująca postać w grze.

Klasa dostarcza podstawowe właściwości i mechanizmy wspólne dla wszystkich postaci w grze, w tym:

  • Statystyki postaci (życie, atak, obrona itp.)
  • Mechanikę obrażeń i leczenia
  • System modyfikatorów i efektów
  • Zarządzanie umiejętnościami

Klasa jest dziedziczona przez PlayerCharacter, EnemyCharacter i inne typy postaci.

Aktualizuje wszystkie modyfikatory statystyk postaci.

Metoda wywołuje metodę Tick() dla wszystkich statystyk postaci, co powoduje:

  • Zmniejszenie czasu trwania modyfikatorów czasowych

Wartość obrony magicznej, która redukuje otrzymywane obrażenia magiczne. Wartość jest obliczana z uwzględnieniem modyfikatorów.
