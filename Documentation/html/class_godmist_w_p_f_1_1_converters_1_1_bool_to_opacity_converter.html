Detailed Description

Konwerter, który zamienia wartość logiczną na wartość przezroczystości (opacity).

Umożliwia ustawienie różnych wartości przezroczystości dla wartości true i false. Format parametru: "wartość_dla_true:wartość_dla_false" (np. "1.0:0.5").
