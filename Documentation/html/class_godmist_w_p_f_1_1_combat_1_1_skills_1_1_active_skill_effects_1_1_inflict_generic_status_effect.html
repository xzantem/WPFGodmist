Detailed Description

Efekt umiejętności, który nakłada na cel ogólny efekt statusowy, taki jak ogłuszenie czy zamrożenie.

Umożliwia nałożenie różnych efektów, które mogą wpływać na możliwości akcji celu. Efekty te zazwyczaj blokują określone akcje na określony czas.
