Detailed Description

ViewModel reprezentujący pojedyncze pomieszczenie w widoku lochów.

Zapewnia dane i logikę dla widoku pokoju, w tym zarządzanie stanem widoczności, aktualizację wyświetlanych informacji oraz powiadamianie o zmianach właściwości.
