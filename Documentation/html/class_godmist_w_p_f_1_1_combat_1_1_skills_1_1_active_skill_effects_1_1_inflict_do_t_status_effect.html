Detailed Description

Efekt umiejętności, który nakłada na cel efekt obrażeń odnoszonych z upływem czasu (DoT).

Może powodować różne typy obrażeń, takie jak krwawienie, zatrucie czy oparzenia. Efekt działa przez określoną liczbę tur i zadaje obrażenia na początku każdej tury.
