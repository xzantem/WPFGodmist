\doxysubsection{Detailed Description}
Główna klasa aplikacji odpowiedzialna za konfigurację i inicjalizację systemów gry. 

Klasa \doxylink{class_godmist_w_p_f_1_1_app}{App} pełni następujące funkcje\+:


\begin{DoxyItemize}
\item Dziedziczy po klasie Application, zapewniając punkty wejścia dla aplikacji WPF 
\item Zarządza cyklem życia aplikacji 
\item Inicjalizuje i konfiguruje główne systemy gry 
\item Obsługuje globalne wyjątki i błędy 
\end{DoxyItemize}

Zawiera metody do zarządzania zasobami, ustawieniami i konfiguracją aplikacji.
