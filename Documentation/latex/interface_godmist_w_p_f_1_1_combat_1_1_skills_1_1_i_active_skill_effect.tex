\doxysubsection{Detailed Description}
Interfejs definiujący efekt umiejętności aktywnej w systemie walki. 

Każdy efekt umiejętności musi implementować tę klasę, aby mógł być używany w systemie walki. Efekty są serializowane przy użyciu konwertera Active\+Skill\+Effect\+Converter.
